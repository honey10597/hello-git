1. git init :-// to create a new repository. 

2. git status :-// List the files you've changed and those you still need to add or commit.

3. git commit :-//  to save in repository.

4. git add . :-//to track all files.

5. git add filename.extension :-// to track particular file.

6. git config :-//o store the configuration for particular repository.
      (a) git config --local :-// for a particular oroject.
      (b) git config --global :-// for single user to all selected user.
      (c) git config --system :-// for all system in a project.

7. git config --local user.name "honey" :-//to give name.

8. git config --local user.name :-// to check/show name.

9. git config --local user.email "abc@gmail.com" :-// to give email.

10. git config --local user.email :-// to display a name.

11. git commit -m "message" :-// save with a message in repository.

12. git remote add origin :-// to send data from local repository to online repository.

13. git remote show origin :-//tells us the place where our code is stored.

13. git clone :-// copies an existing git repository.
